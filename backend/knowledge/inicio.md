# Início
Este documento visa explicar o funcionamento básico do Comercium, com exemplos sobre como criar conta, realizar login, acessar o sistema, entre outros.

## Criação de Conta
O primeiro passo para criar uma conta é acessar o link do [Comercium](https://comercium.com.br), depois, basta seguir os seguintes passos:
1. Clique em "Começar agora"
2. Na tela que abrir, insira seus dados e de sua loja.
    2. 1. E-mail: o e-mail utilizado para acesso ao sistema. Também é o mesmo e-mail que receberá futuras atualizações de funcionalidades ou termos de uso.
    2. 2. Telefone: telefone de contato da loja. Pode ser utilizado para receber notificações de atualizações ou contatos do Comercium com sua loja.
    2. 3. Código da Loja: identificador único da loja no Comercium. Deve ser escrito em letras minúsculas.
    2. 4. Senha: senha de acesso ao sistema. Certifique-se de usar uma senha forte e que não é utilizada em outros lugares.
    2. 5. Documento: CPF do utilizador ou CNPJ da loja. Para emissão de notas fiscais, é necessário cadastrar por CNPJ. Caso seja cadastrado CPF, o documento poderá ser alterado posteriormente para um CNPJ.
3. Realize o login utilizando o e-mail e senha cadastrados
4. Proceda para os manuais de uso do sistema, como os procedimentos de adição de clientes, produtos, abertura de caixa e realização de vendas.

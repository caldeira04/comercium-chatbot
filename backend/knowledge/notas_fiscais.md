# Notas Fiscais

> **Módulo em desenvolvimento.**

A funcionalidade de emissão de notas fiscais ainda não está disponível no
Comercium.

## Preparação

Para utilizar notas fiscais quando o módulo for lançado:

1. Cadastre sua loja com **CNPJ** (pessoa jurídica).
2. Mantenha os dados da loja atualizados (endereço, regime tributário).
3. Cadastre os produtos com **NCM** e **CEST** corretos.

Fique atento às atualizações do sistema para quando este módulo for disponibilizado.
